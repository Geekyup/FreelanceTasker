# FreelanceTasker

**FreelanceTasker** — это удобная платформа для управления фриланс-проектами, задачами и взаимодействия между заказчиками и исполнителями.

## О проекте
— Позволяет публиковать задачи и проекты, брать их в работу, отслеживать прогресс.
— Система профилей пользователей с рейтингом.
— Интуитивный интерфейс для работы с заказами и задачами.

## Основной функционал:
- Регистрация и авторизация пользователей
- Публикация и выполнение задач
- Комментарии и отзывы
- Система прогресса/статусов задач
- Управление профилем

## Технологии
- Python/Django
- Frontend: HTML, CSS, JS 
- SQLite/PostgreSQL (бэкенд БД)

## Как запустить проект локально

- git clone https://github.com/Geekyup/FreelanceTasker.git
- cd FreelanceTasker
- pip install django
- python manage.py migrate
- python manage.py runserver
